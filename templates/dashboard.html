    Quick Start Guide
    Load into Tableau / Any BI Tool:
    1. Download the Full Export above
    2. Unzip — you'll find 4 CSV files + manifest.json
    3. Load each CSV as a data source / data stream
    4. Set relationships: alert_details.msg_id → messages.msg_id, alert_details.city_id → cities.city_id, alert_details.zone_id → zones.zone_id
    5. Filter to real alerts: message_type = "alert" AND is_drill = 0
    6. Start visualizing! Group by city, zone, date, alert_type
    For Salesforce Data Cloud + Tableau Next:
    1. Download the Raw + Formulas export
    2. Create 4 Data Streams (messages, alert_details, cities, zones)
    3. Create DMOs from each stream
    4. Build semantic model with star schema relationships
    5. Create calculated fields using formulas from manifest.json
    6. See TABLEAU_GUIDE.md for detailed walkthrough with tested formulas
